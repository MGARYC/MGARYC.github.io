pm2进程管理与维护

安装

全局安装，简直不能更简单。
npm install -g pm2

目录介绍

pm2安装好后，会自动创建下面目录。看文件名基本就知道干嘛的了，就不翻译了。

命令

  • 查看进程状态
    pm2 list
  • 删除
    pm2 delete name
  • 停止
    pm2 stop all
  • 启动
    pm2 start npm --name "jsyfShopNuxt" -- run start
